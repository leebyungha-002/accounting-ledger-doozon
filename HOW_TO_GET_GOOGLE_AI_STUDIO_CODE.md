# Google AI Studio 프로젝트 코드 가져오기 가이드

## 📋 방법 1: Google AI Studio 웹에서 코드 복사

### 단계별 안내

1. **Google AI Studio 접속**
   - https://aistudio.google.com 접속
   - 로그인 후 프로젝트 선택

2. **코드 확인 방법**
   - 프로젝트 화면에서 코드 편집기 영역 확인
   - 보통 왼쪽 또는 중앙에 코드가 표시됨
   - 전체 코드 선택 (Ctrl+A 또는 Cmd+A)
   - 복사 (Ctrl+C 또는 Cmd+C)

3. **파일 구조 확인**
   - Google AI Studio는 보통 다음 파일들을 포함:
     - `index.html` 또는 `Index.html` - HTML 구조
     - `index.tsx` 또는 `Index.tsx` - React/TypeScript 코드
     - `metadata.json` - 프로젝트 메타데이터
     - 스타일 파일 (CSS)

4. **각 파일별로 복사**
   - 각 파일 탭을 클릭하여 내용 확인
   - 파일별로 복사하여 텍스트 파일로 저장

---

## 📋 방법 2: 브라우저 개발자 도구 사용

### 단계별 안내

1. **Google AI Studio 프로젝트 열기**
   - 프로젝트를 실행 상태로 열기

2. **개발자 도구 열기**
   - `F12` 키 누르기
   - 또는 `Ctrl+Shift+I` (Windows) / `Cmd+Option+I` (Mac)

3. **Sources 탭 확인**
   - 개발자 도구에서 "Sources" 탭 클릭
   - 왼쪽 파일 트리에서 프로젝트 파일 확인
   - 파일 클릭하여 코드 확인 및 복사

4. **Network 탭에서 API 호출 확인**
   - "Network" 탭에서 API 호출 방식 확인
   - 특히 Gemini API 호출 방식을 확인

---

## 📋 방법 3: 프로젝트 내보내기/다운로드

### Google AI Studio에서 제공하는 경우

1. **프로젝트 설정 확인**
   - 프로젝트 화면에서 설정 아이콘 클릭
   - "Export" 또는 "Download" 옵션 확인

2. **코드 복사**
   - 일부 Google AI Studio 버전에서는 코드를 직접 복사 가능

---

## 📋 방법 4: 로컬에 저장된 파일 확인

### 이미 다운로드했거나 로컬에 있는 경우

1. **다운로드 폴더 확인**
   - `C:\Users\USer\Downloads`
   - 또는 문서 폴더 확인

2. **프로젝트 이름으로 검색**
   - Windows 탐색기에서 프로젝트 이름으로 검색
   - 예: "accounting", "ledger", "analysis" 등

3. **파일 확장자로 검색**
   - `.html`, `.tsx`, `.ts`, `.jsx`, `.js` 파일 검색

---

## 📋 방법 5: 스크린샷 또는 화면 공유

### 코드를 직접 복사하기 어려운 경우

1. **스크린샷 촬영**
   - 코드 영역의 스크린샷 촬영
   - 여러 장 촬영하여 전체 코드 커버

2. **화면 녹화**
   - 코드를 스크롤하며 녹화
   - 나중에 코드를 확인
