**질문 2: Git Bash 창에 무엇이 보이나요?**

```

User@DESKTOP-V1M97UH MINGW64 ~/Documents/accounting-ledger-analysis
```

→ 에러 메시지 전체를 복사해주세요.

## Step 2: 포트 확인

Git Bash에서 **정확히 이 명령어를 실행**해주세요:

```bash
netstat -ano | findstr :8080
```

**결과가 없으면**: 8080 포트가 비어있습니다 (문제!)
**결과가 있으면**: 8080 포트가 사용 중입니다 (정상)

---

## Step 3: 다른 포트로 시도

혹시 포트 문제일 수 있으니 **다른 포트**로 실행해봅시다:

### Git Bash에서:

```bash
# 현재 서버 중지
Ctrl + C

# 포트 3000으로 실행
npm run dev -- --port 3000
```


이게 작동하나요?
