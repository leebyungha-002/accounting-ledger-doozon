# Cursor 세션 참고사항

## 중요: 작업 종료 전 커밋 확인

**사용자 요청사항:**
- 작업 종료 전에 항상 변경사항 커밋 여부를 확인해야 함
- 질문 형식: "작업을 종료하시기 전에 변경사항을 커밋하시겠습니까?"

**주의사항:**
- AI 어시스턴트는 세션 간 대화 내용을 기억하지 못함
- 하지만 이 파일을 참조하여 다음 세션에서도 동일한 요청사항을 확인할 수 있음
- 사용자가 "종료", "끔", "전원 off" 등의 표현을 사용하면 커밋 확인 질문하기
