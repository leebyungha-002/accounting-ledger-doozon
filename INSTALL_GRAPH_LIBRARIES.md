# 그래프 다운로드 기능을 위한 라이브러리 설치

그래프를 이미지나 PDF로 다운로드하려면 다음 라이브러리를 설치해야 합니다.

## 설치 방법

프로젝트 디렉토리에서 다음 명령어를 실행하세요:

```bash
npm install html2canvas jspdf
```

또는 PowerShell에서:

```powershell
cd Documents\accounting-ledger-doozon
npm install html2canvas jspdf
```

## 설치 후

라이브러리 설치 후 개발 서버를 재시작하세요:

```bash
npm run dev
```

## 기능

- **이미지 다운로드**: 그래프를 PNG 이미지로 다운로드
- **PDF 다운로드**: 그래프를 PDF 파일로 다운로드

라이브러리가 설치되지 않은 경우, 버튼을 클릭하면 설치 안내 메시지가 표시됩니다.
