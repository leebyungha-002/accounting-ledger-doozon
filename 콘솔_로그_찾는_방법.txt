콘솔에서 로그 찾는 방법
====================

방법 1: 필터 사용 (가장 간단)
----------------------------
1. 개발자 도구 열기: F12
2. Console 탭 클릭
3. 필터 입력창에 다음 중 하나 입력:
   - "월별합계" (한글)
   - "월별 합계" (띄어쓰기 포함)
   - "🔵" (이모지)
   - "shouldShow"
   - "generalDrilldownShowMonthly"

방법 2: 로그 레벨 필터
---------------------
1. Console 탭에서 필터 아이콘 클릭
2. "Info" 또는 "All levels" 선택
3. "Warnings"도 체크 (경고 메시지 확인)

방법 3: 검색 기능 사용
---------------------
1. Console 탭에서 Ctrl + F (검색)
2. 다음 키워드로 검색:
   - "월별합계"
   - "표시 조건"
   - "generalDrilldownShowMonthly"

방법 4: 콘솔 지우고 다시 테스트
-------------------------------
1. Console 탭에서 🚫 아이콘 클릭 (콘솔 지우기)
2. 일반사항분석에서 계정명 또는 차변/대변 금액 클릭
3. 새로 나타나는 로그만 확인

찾아야 할 로그 메시지:
---------------------
✅ "🔵 [계정명 클릭]:" 또는 "🔵 [차변 클릭]:" 또는 "🔵 [대변 클릭]:"
✅ "✅ [계정명 클릭] generalDrilldownShowMonthly를 true로 설정했습니다."
✅ "🔵 [월별합계] 표시 조건 확인:"
✅ "⚠️ [월별합계] 표시되지 않음" (만약 표시되지 않으면)

로그가 전혀 보이지 않으면:
-------------------------
- 브라우저를 완전히 닫고 다시 열기
- Ctrl + Shift + R (강력 새로고침)
- 개발자 도구를 닫았다가 다시 열기 (F12)
